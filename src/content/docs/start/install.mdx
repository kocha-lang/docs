## O'rnatish

Agar siz Windows x64 tizimi sohibi bo'lsangiz, siz [Githubdan](https://github.com/kocha-lang/kocha)
oxirgi interpretatorni, ya'ni `kocha.exe` ni yuklab olsangiz yetarli.

Yoki, shunchaki [Deno](https://deno.com/) yordamida ishlatsangiz bo'ladi.


<Steps>

1. Deno ni o'rnating

2. Git ni o'rnating


3. Kocha repozitoriyasini yuklab oling:
   
   ```bash
   git clone https://github.com/kocha-lang/kocha.git
   ```

4. Sinab koring

   ```bash
   cd kocha
   deno run main.ts
   # Bu sizga Kocha dasturlash tilining repl-ini ochib berishi kerak.
   ```

</Steps>


## Ishlatish
Keling, siz bilan *"salom dunyo"* dasturini yozamiz. \
Uning uchun `salom.kocha` faylini yarating, va ichiga quyidagi kodni yozing.


<Code code={'korsat("Salom Dunyo!");'} lang="Kocha Tili" title={"salom.kocha"} mark={['file', 'CMS']} />

Shunchaki `kocha.exe` yoki `deno run main.ts` qilib ochsangiz, siz Kocha lang-ni
[repl-iga](https://en.wikipedia.org/wiki/Read%E2%80%93eval%E2%80%93print_loop) kirasiz.

Biz esa, `salom.kocha` faylini interpretatsiya qilmoqchimiz. 
Shuning uchun `kocha.exe` yoki `deno run main.ts` oldiga `salom.kocha` faylimizni joylashuvini kiritishimiz kerak.

```bash
deno run --allow-read main.ts salom.kocha # deno bilan
# ---
kocha.exe salom.kocha # kocha.exe bilan
```
Yozganimiz ishashi uchun `kocha.exe` yoki `main.ts`, va `salom.kocha` bir direktoriyada bo'lishi kerak.
