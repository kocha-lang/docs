Kocha Lang bilan qulay ishlash uchun sizning ko'dingizga syntax highlight
kerak bo'ladi. Ko'cha Tilini VSCode foydalanuvchilari uchun sintaksis [qo'shimchasi](https://github.com/kocha-lang/extension/)
mavjud.

## VSCode
<Steps>
1. Qo'shimchaning repozitoriyasini yuklab oling
   ```bash
   git clone https://github.com/kocha-lang/extension.git kocha-lang-extension
   ```

2. VSCode qo'shimchalari saqlanadigan direktoriyaga boring
   ```bash
   # windows uchun
   # YourUser o'rniga oâ€˜zingizni windows useringiz
   cd "C:\Users\YourUser\.vscode\extensions"
   ```

3. Shu yerga yuklab olingan `kocha-lang-extension` direktoriyasini qo'ying.
4. Tayyor! VS Code-ni ochib tekshirib koring.
</Steps>
